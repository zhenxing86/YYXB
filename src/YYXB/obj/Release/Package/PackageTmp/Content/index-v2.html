【问题指导】孩子不喜欢读书怎么办？

我们都渴望有一个爱阅读的孩子，但是有时当我们满怀信心的拿出书给孩子看的时候，却发现孩子并不喜欢阅读，这该怎么办呢？我们可以在陪伴孩子阅读时增加一些技巧，让孩子慢慢喜欢上阅读，成为一个爱阅读的宝宝。
